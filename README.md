# park-file-manager
Web File Manager

웹기반 파일 관리자

현재 사용 기능
- 파일 업로드
- 파일 다운로드
- 홈 이동
- 상위폴더 이동
- 하의폴더 이동

- file upload
- file download
- go home
- go parent directory
- go child directory


개발 환경
- IntelliJ 프로젝트
- Gradle
- Spring Boot 1.4
- Spring Web 4.3.2
- Java 1.8
- jquery 3.1
- Tomcat 8.5.4
- War deploy
